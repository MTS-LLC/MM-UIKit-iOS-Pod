## Installation

To install, simply add the following line to your Podfile:

```ruby
pod 'UIKit-iOS-Pod', :git => 'https://github.com/MTS-LLC/MM-UIKit-iOS-Pod.git'
```

For installing the first time run:
```ruby
pod install
```

To get the latest version of the pod run:
```ruby
pod update
```
